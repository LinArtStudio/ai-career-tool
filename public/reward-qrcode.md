# 微信赞赏码配置说明

## 使用方式

1. 打开微信 → 我 → 服务 → 收付款 → 赞赏码
2. 生成赞赏码并保存图片
3. 将图片保存为 `reward-qrcode.png` 放在 `public/` 目录
4. 用户付费后，手动开通对应功能权限

## 付费流程

### 用户操作
1. 点击"购买"按钮
2. 显示赞赏码和付费说明
3. 用户扫码支付
4. 用户截图发送到客服邮箱/微信
5. 人工审核并开通权限

### 权限管理
- 免费用户：每天3次简历诊断、3次JD匹配、1道模拟面试
- 付费用户：无限次使用所有功能
