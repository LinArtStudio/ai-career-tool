### 第1步：整理工作内容（5分钟）

**Prompt**：
```

输出格式：
1. 项目A：完成了XX，成果是XX
2. 项目B：推进了XX，进展是XX
3. 项目C：解决了XX，效果是XX
```

**效果**：
- AI自动分类工作内容
- 突出关键成果
- 生成结构化输出

### 第2步：生成周报框架（5分钟）

**Prompt**：
```

输出格式：
1. 本周工作总结（3-5条）
2. 关键成果（数据化）
3. 遇到的问题及解决方案
4. 下周工作计划
5. 需要的支持和资源
```

**效果**：
- AI生成完整周报框架
- 自动数据化成果
- 结构清晰专业

2. 数据化展示效果

**效果**：

### 技巧1：给AI清晰的上下文

**错误**：
```

```

**正确**：
```
请帮我写一份产品经理的周报，本周主要完成了：
1. 完成了AI客服功能的PRD
2. 推进了用户增长项目
3. 解决了3个线上bug
```

### 技巧2：分步骤生成

**错误**：
```

```

**正确**：
```
先整理工作内容 → 再生成框架 → 最后优化表达
```

### 技巧3：人工审核和优化

**AI生成的内容需要人工审核**：
- 检查数据是否准确
- 补充遗漏的工作
- 优化表达方式
- 添加个人风格
