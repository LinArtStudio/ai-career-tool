## 🎯 Supabase配置指引

### 你需要做的3件事（约15分钟）

**第1步：注册Supabase账号（5分钟）**
1. 访问 https://supabase.com
2. 点击 "Start your project"
3. 使用GitHub账号登录
4. 创建新项目：
   - Project name: `ai-career-tool`
   - Database Password: 设置强密码（记住！）
   - Region: Northeast Asia (Tokyo)

**第2步：获取API信息（2分钟）**
1. 进入项目后，点击 Settings → API
2. 复制两个值：
   - Project URL: `https://xxx.supabase.co`
   - anon/public key: `eyJxxx...`

**第3步：执行数据库Schema（5分钟）**
1. 点击左侧 SQL Editor
2. 点击 New query
3. 复制 `supabase/schema.sql` 的内容并执行

**第4步：配置环境变量（3分钟）**
1. 在Vercel添加环境变量
2. 在阿里云服务器添加环境变量
3. 重新部署应用
