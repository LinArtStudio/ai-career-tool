# 产品定位文档 v2.0

**更新时间**：2026年6月24日  
**定位调整**：从"AI产品经理求职训练平台"扩展为"AI时代求职训练平台"

## 一、产品定位

### 1.1 一句话定位

> **AI时代求职训练平台**
> 帮助所有求职者掌握AI时代必备技能，提升求职竞争力

### 1.2 覆盖范围

| 用户类型 | 岗位示例 | 核心需求 |
|----------|----------|----------|
| **AI相关岗位** | AI产品经理、AI工程师、AI训练师、提示词工程师 | AI技术深度、项目经验、面试准备 |
| **AI+传统岗位** | 产品经理+AI、运营+AI、设计师+AI、市场+AI | AI工具使用、效率提升、转型路径 |
| **传统岗位** | 产品经理、运营、设计师、市场、销售 | AI基础认知、工具使用、竞争力提升 |

### 1.3 核心价值主张

**对于AI相关岗位**：
- 系统准备AI岗位面试
- 掌握RAG、Agent、Prompt Engineering等技术
- 积累AI项目经验
- 获得大厂offer

**对于AI+传统岗位**：
- 用AI工具提升效率10倍
- 掌握AI时代必备技能
- 从"被替代者"变成"驾驭者"
- 获得更好的职业发展

**对于传统岗位**：
- 了解AI对行业的影响
- 掌握基础AI工具使用
- 提升求职竞争力
- 避免被AI替代

## 二、用户画像

### 2.1 用户类型1：AI原生型（AI-Native）

**用户特征**：
- 目标：获得AI相关岗位offer
- 背景：应届生、初级职场人、技术背景转型者
- 痛点：技术认知不足、缺乏实战经验、面试准备错误

**核心需求**：
- AI技术深度理解
- AI项目实战经验
- 面试准备和题库
- 简历优化

**对应功能**：
- JD智能匹配诊断（AI岗位专项优化）
- AI模拟面试（150+大厂真题）
- 面试问题预测（技术深度题）
- 项目经验包装

### 2.2 用户类型2：AI赋能型（AI-Empowered）

**用户特征**：
- 目标：用AI提升现有工作效率
- 背景：3-5年经验的产品、运营、设计师等
- 痛点：技能焦虑、转型迷茫、效率瓶颈

**核心需求**：
- AI工具使用培训
- 效率提升方法
- 转型路径指导
- 职业规划

**对应功能**：
- AI职业竞争力测评
- AI工具效率提升课程
- 行业解决方案
- 职业规划

### 2.3 用户类型3：传统岗位型（Traditional）

**用户特征**：
- 目标：了解AI、提升竞争力
- 背景：所有传统岗位求职者
- 痛点：对AI不了解、担心被替代

**核心需求**：
- AI基础认知
- 工具使用入门
- 行业趋势了解
- 竞争力提升

**对应功能**：
- AI职业竞争力测评（入门级）
- AI基础知识课程
- 行业AI应用案例
- 简历AI能力展示

## 九、风险与应对

**风险**：定位太宽，失去焦点
**应对**：保持核心功能聚焦，扩展覆盖范围

### 9.3 用户风险

**风险**：用户需求分散，难以满足
**应对**：分层服务，满足不同用户需求
