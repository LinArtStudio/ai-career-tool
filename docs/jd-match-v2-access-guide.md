## 🛠️ 如果无法访问

### 问题1：浏览器显示"不安全"

**原因**：自签名SSL证书不被浏览器信任

**解决方案**：
1. 点击浏览器地址栏的"不安全"提示
2. 点击"证书"或"高级"
3. 点击"继续访问（不推荐）"
4. 或者使用HTTP方式访问：http://114.55.110.19:3000/jd-match-v2

### 问题2：页面加载缓慢

**原因**：服务器响应需要时间

**解决方案**：
1. 等待10-30秒
2. 刷新页面
3. 清除浏览器缓存

### 问题3：显示404或空白页

**原因**：可能是浏览器缓存

**解决方案**：
1. 强制刷新：Ctrl + F5（Windows）或 Cmd + Shift + R（Mac）
2. 清除浏览器缓存
3. 使用隐身模式访问

### 问题4：API调用失败

**原因**：LLM API配置问题

**解决方案**：
1. 检查服务器环境变量
2. 确认LLM API Key有效
3. 查看服务器日志：`pm2 logs ai-career`
