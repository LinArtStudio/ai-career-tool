## 一、产品定位

### 1.1 一句话定位

> **AI产品经理求职训练平台**
> 帮助应届生和转型者，系统准备AI产品经理面试，从简历到面试全流程

### 1.2 定位说明

**目标用户**：
- **主要用户**：2026届应届生，求职AI产品经理岗位
- **次要用户**：传统产品经理转型AI方向
- **潜在用户**：对AI产品感兴趣的学生和职场人

**核心价值**：
1. **面试问题预测**：基于150+大厂真题，预测AI产品经理面试问题
2. **技术深度指导**：帮助理解RAG、Agent、Prompt工程等技术选型
3. **项目经验包装**：帮助量化AI产品项目成果

**差异化优势**：
- ✅ **独家功能**：面试问题预测（竞品都没有）
- ✅ **技术深度**：帮助理解AI技术选型逻辑
- ✅ **中国市场**：针对大厂面试特点优化

## 二、目标用户画像

### 2.1 主要用户：应届生求职AI产品经理

**用户特征**：
- 年龄：20-25岁
- 学历：本科/硕士，计算机/产品相关专业
- 状态：在校或应届毕业
- 目标：进入大厂（字节、腾讯、阿里、百度等）

**核心痛点**：
1. **技术认知不足**：不懂RAG、Agent、Prompt工程等技术选型
2. **缺乏实战经验**：没有真实的AI产品项目经历
3. **面试准备错误**：只背题不理解，缺乏"判断力"
4. **简历不会写**：不知道如何量化AI产品项目成果

**用户场景**：
```
场景1：准备字节AI产品经理面试
- 输入：字节AI产品经理JD
- 输出：面试问题预测 + 技术要点 + 回答思路

场景2：优化AI产品项目简历
- 输入：简历内容 + 目标岗位JD
- 输出：匹配度评分 + 优化建议 + 项目量化建议

场景3：模拟AI产品经理面试
- 输入：目标公司 + 岗位
- 输出：5道定制题 + AI评估 + 面试官视角反馈
```

### 2.2 次要用户：转型者

**用户特征**：
- 年龄：25-35岁
- 背景：传统产品经理、运营、技术等
- 状态：在职，想转型AI方向
- 目标：获得AI产品经理岗位

**核心痛点**：
1. **技术门槛**：AI技术知识不足
2. **经验转换**：现有经验如何迁移到AI领域
3. **竞争压力**：与应届生和AI背景候选人竞争

## 三、核心功能定位

### 3.1 功能矩阵

| 功能 | 定位 | 目标用户 | 差异化 |
|------|------|----------|--------|
| **JD智能匹配诊断** | 核心功能 | 所有用户 | ✅ 多维度评分 + 面试问题预测 |
| **AI模拟面试** | 核心功能 | 所有用户 | ✅ 150+大厂真题 + AI评估 |
| **简历诊断** | 基础功能 | 所有用户 | 5维度评分 + 优化建议 |
| **职业规划** | 辅助功能 | 应届生 | 3条路径 + 学习计划 |
| **JD匹配分析** | 辅助功能 | 所有用户 | 关键词匹配 + 差距分析 |
| **AI求职信** | 增值功能 | 付费用户 | 定制生成 |

### 3.2 核心功能详解

#### JD智能匹配诊断（v2.0）

**功能定位**：AI产品经理求职的核心工具

**核心价值**：
1. **多维度匹配评分**：技能、经验、教育、软技能独立评分
2. **精准差距分析**：关键差距、中等差距、轻微差距分类
3. **面试问题预测**（独家）：行为、技术、情景、压力面试问题
4. **针对性简历优化**：关键词、经历描述、技能展示优化

**用户价值**：
- 知道自己与目标岗位的匹配度
- 知道需要弥补哪些差距
- 提前准备面试问题
- 优化简历提升通过率

#### AI模拟面试

**功能定位**：面试能力训练工具

**核心价值**：
1. **真题库**：150+大厂面试题，覆盖18家公司
2. **AI评估**：评分 + 优点 + 不足 + 优化建议
3. **面试官视角**：从面试官角度给出反馈
4. **进度追踪**：练习次数、平均分、连续天数

**用户价值**：
- 熟悉面试题型和难度
- 提升面试表达能力
- 量化学习进度
- 增强面试信心
